# RAIN Anonymous Artifact

This directory is an anonymized release copy of the proof-repair agent and its evaluation runners. Evaluation code is grouped under `Evaluation/`.

Start with `INSTALL.md` for artifact setup and final validation tests. Real benchmark runs live under `Evaluation/`; the individual evaluation folders include dedicated README files with suite-specific commands and options.

Generated artifacts, local logs, binary sentence splitters, result folders, Python caches, .NET build output, IDE state, and local configuration files are intentionally omitted.
