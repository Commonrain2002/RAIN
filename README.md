The directories `AnonymousCobbleStone/` and `AnonymousPALM/` contain our reproduced implementations of CobbleStone and PALM, respectively, and `AnonymousRAIN/` contains our proposed system. For environment setup, please follow the `INSTALL.md` instructions provided in each directory. The directories `Result/`, `CoqStoqStep/`, and `CoqGymStep/` hold evaluation artifacts and step-count statistics; run `python3 Scripts/evaluation_numbers.py` from the repository root to recompute the Evaluation-section numbers from those files.
